Stay Tuned for more details on LIVE! ART && CODE, Coming Fall 2023!

This Fall 2023, CMU's Frank-Ratchye STUDIO for Creative Inquiry will present LIVE! ART && CODE — an event series exploring liveness and performance with creative technologies. The STUDIO's long-running Art && Code event series is concerned with democratizing the cultural and creative potentials of emerging technologies.

Please direct questions to studio-info@andrew.cmu.edu.
